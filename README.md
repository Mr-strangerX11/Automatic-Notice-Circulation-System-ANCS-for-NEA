# utomatic-Notice-Circulation-System-ANCS-for-NEA
It is an Full-stack internal system for uploading a notice once and automatically circulating via email, SMS (mock Sparrow/NTC), push (FCM), and dashboards. Implements JWT auth, RBAC for 5 roles, tracking (seen/unseen/downloads), delivery logs, admin/department dashboards, and a digital noticeboard.
